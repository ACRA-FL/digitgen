## Installation

### 1. From PIP

$`pip install ImgAnn`

### 2. From github
clone the codebase from GitHub
```
$ git clone https://github.com/ACRA-FL/digitgen.git
```
build the library
```
$ python setup.py bdist_wheel sdist
$ pip install -e .
```
